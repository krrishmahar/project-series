Chat Application in JAVA

This project is a real-time chat application developed in Java, utilizing Maven as the build tool and WebSocket for enabling bi-directional communication between clients and the server. This chat application is a fundamental exercise in understanding and implementing real-time communication in Java, leveraging modern tools and technologies.

Backend: The server-side of the application is built using Java and the WebSocket API. The server is responsible for handling incoming WebSocket connections, broadcasting messages to all connected clients, and managing the overall communication logic.

Frontend: The client-side is typically a web application that connects to the WebSocket server. It provides the interface for users to send and receive messages.

Dependencies: Maven is used to manage dependencies such as Java-WebSocket, ensuring that all required libraries are included and up-to-date.

The Network Scanner project, part of the "project-series" repository, focuses on developing a tool for scanning and analyzing network devices. This project, built during the second week of the networking series, demonstrates practical applications of network programming concepts and provides a foundation for understanding network security and diagnostics.


Key Feature

Device Discovery: The Network Scanner can detect devices connected to a local network. This feature is crucial for network administrators to identify all active devices within a subnet.

Port Scanning: It includes functionality to scan open ports on the discovered devices. This helps in identifying potential vulnerabilities, as open ports can be entry points for unauthorized access.

Ping Sweep: The scanner can perform a ping sweep, sending ICMP echo requests to a range of IP addresses to determine which hosts are up. This is useful for mapping out network topology and identifying active devices.

Multi-threading: To enhance performance, the tool uses multi-threading for simultaneous scanning of multiple devices and ports, significantly reducing the time required for a complete network scan.
