User-Friendly Interface:

The application includes a simple, intuitive user interface that allows users to join the chat, send messages, and view messages from other users in real-time. Somplicity at Peak

This project is a real-time chat application developed in Java, utilizing Maven as the build tool and WebSocket for enabling bi-directional communication between clients and the server. This chat application is a fundamental exercise in understanding and implementing real-time communication in Java, leveraging modern tools and technologies.

Backend: The server-side of the application is built using Java and the WebSocket API. The server is responsible for handling incoming WebSocket connections, broadcasting messages to all connected clients, and managing the overall communication logic.

Frontend: The client-side is typically a web application that connects to the WebSocket server. It provides the interface for users to send and receive messages.

Dependencies: Maven is used to manage dependencies such as Java-WebSocket, ensuring that all required libraries are included and up-to-date
